The Psychologist’s data toolbox 2.0

Matti Vuorre

2016-09-02

Welcome to the Psychologist’s data toolbox 2.0. This website contains educational materials on using R and RStudio for effective data wrangling and visualization. In addition to introductory materials, there are tutorials for Psychology-specific data-analytic scenarios. Get started by clicking on the menu items above.

For first-time R users, I recommend reading the sections of this website in order (from left to right in the above navigation bar.) First, you should read the introduction, and then get to know the tools. When you are familiar with the RStudio environment, common steps in the data-analytic workflow are covered in their typical order. If you already know how to import data, for example, skip that section and move right on to the next topics.

Following data visualization, you would typically move to modelling the data, but this topic is outside the scope of these tutorials. I recommend reading a good textbook (e.g. Kruschke, 2014; Maindonald & Braun, 2010; McElreath, 2016).

References

Kruschke, J. K. (2014). Doing bayesian data analysis: A tutorial introduction with r (2nd Edition). Burlington, MA: Academic Press.

Maindonald, J., & Braun, W. J. (2010). Data analysis and graphics using r: An example-based approach (Vol. 10). Cambridge University Press.

McElreath, R. (2016). Statistical rethinking: A bayesian course with examples in r and stan. CRC Press.
